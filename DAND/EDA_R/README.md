This is the project assignment of the module “Exploratory Data Analysis” in the Data Analyst NanoDegree program. This analysis was written in R using the IDE “R Studio”. This is based on ATO data made available during the NSW GovHack 2017. The files in this directory are:
	* ato_govhack.html - HTML output of the rmd file
	* ato_govhack.rmd - RMD file containing project
	* Postcode_Ranges.doc - Document containing postcode ranges of states
	* atoabsgovhack2017.xlsx - Spreadsheet containing the data used for analysis.
	* individual-tax-return-2015.pdf - PDF file of tax file application which contains information on how to calculate certain components of tax.
